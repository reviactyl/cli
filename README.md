# Reviactyl CLI
Made installation easier.

## Features
- Installs Reviactyl Panel
- Uninstall Reviactyl Panel
- Upgrade Reviactyl Panel

```
bash <(curl -s https://raw.githubusercontent.com/reviactyl/cli/refs/heads/main/install-cli.sh)
```
